Linux vs Windows 11

Jul 25th, 2021

We’ve all heard the news; Microsoft has created a new version of windows version 11 but what does this mean well let me tell you. I’m not a windows user myself I use kali and ubuntu on all of my machines and I find it very great, it runs fast, I can customize it to anything I want and much more.

But windows is kind of a target for me as a hacker we write malware for mostly windows because it’s the most used operating system used by millions of users all across the world. But windows 10 is coming to an end by 2025 Microsoft is going to stop supporting the operating system.

But you may be asking yourself: Why?!


Well, Microsoft has said that windows 11 won’t run on older generation of intel CPU it will only run on from the 9th generation and further. And this will mean that in 2025, 50% of all windows users will have no support anymore.

So what are your options to stay supported:

  • Switch to Linux
  • Buy a new laptop or desktop
  • Stick with windows 10


    • Let’s start with the first one Switch to Linux

    Linux is a wonderful, but you may be confused which distro or distribution you should choose well there are a couple of operating to choose from. From ubuntu to popOs, from manjaro to arch. Loads of options.

There’s a shortage of chips at the moment and most laptops which you can get for cheap are now double or triple as expensive than the original price. But if you have family or friends that are not using you could ask them for it and use that if it’s still supported for windows 11.


    3 stick with windows 10 Luckily we’re still in 2021 and windows 10 is still supported but you should really start planning one of these options for the future.
