WI-FI is a vulnerable system it’s great for connecting wirelessly to your smartphone and laptop but it can also be used for malicious activity such as planting malware or a virus, changing content on the fly and even spying.


Basic Cracking
All hackers know that Wi-Fi networks are never secure even if you use the highest encryption possible and have the longest password you can generate. But there are numerous ways to crack and gain access to a network without knowing the password, and it’s the fastest way to crack a network usually in like 15-20 seconds.


WPS Pixie dust attack or brute force attack
This is one of my favorite attacks for cracking networks WPS stands for (Wifi Protected Setup) it’s a handy but vulnerable feature that every router has, it’s handy for using an 8-digit code to connect to the network, but it can be used against you let me explain.


How it works?
A client wants to connect to the network, using WPS you use an 8-digit code that you type into the device and the router authenticates the pin and responds back with password of the network. But we as hacker if we have an antenna in monitor mode and a hacker machine that starts generating wps pins and sending them to the router and if the right pin was sent the router will respond with password to connect to the network.


After this you can go to the post-connection attacks that could be scanning the network, writing malware, looking for vulnerabilities, etc.
