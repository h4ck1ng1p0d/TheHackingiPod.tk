BetterDiscord

Aug 31st, 2021

Everybody must have heard about Discord the messaging and calling platform made for gamers. It’s a great platform, I personally use it but rarely though but not since a couple of days ago…

BetterDiscord
A couple of days ago I came up with an thought about discords security, let me explain there’s this TV show called Hunted which as the title says it’s a game where around 10 people need to stay out of the hand of the hunters so a criminal from the police, and seeing how these people stay out of the hands of the hunters and figure out different ways to communicate with outsiders to get them places. And 2 guys who are hardcore gamers used discord and I started thinking I was like WOW. That’s actually pretty good but then I thought oh shit!!. Discord in not encrypted, so I started researching if there was a way to encrypt a chat communication on discord and then I came across this plugin which only works with BetterDiscord.


What can you do? (Themes and plugins)
After installation I went the official website and was amazed on how many themes and plugins you can use to change up the look and feel of your discord client. But the best thing of all is that it’s based on really simple programming languages for example: Plugin: every plugin is written using JavaScript so essentially everything you can do with JavaScript you can do it. Themes: Also fun for the web developers, the theme builder uses CSS so if you’re good at styling using CSS your are in luck.


Favorite Plugins
My absolute favorite plugin is the read all button. I’ve joined lots of servers in the past years and having this button so I don’t have to click ‘read’ for every server. But another thing is the encryption, encrypting every message with SHA-256 encryption and making mass surveillance difficult so the company can’t see what I’m doing.
