Windows 10 Encryption Sucks!!

Feb 6th, 2022

I recently got a call from a friend telling me that she got locked out of her Windows 10 machine.
And after doing some research i’ve found that resetting your windows 10 password is pretty f*cking easy.
Here’s a couple of ways:

How to reset the password on windows 10 (MS account)

  1. go to https://account.live.com/password/reset
  2. Enter your account email you used
  3. Enter new password
  4. Try again loging in


How to reset the password on windows 10 (local acccount)
Using Recovery Mode (This doesn't work on all machines, if it doesn't move to the next method)

  1. On the login screen --> Recovery mode Shift + Reboot
  2. Once a blue screen appears then click on Troubleshoot
  3. Next, Advanced Options
  4. Then, Command Prompt (CMD)
  5. Locate disk, enter DISKPART
  6. then exit and enter (disk letter):
  7. change directory cd C:\Windows\System32\
  8. backup utilman.exe & cmd.exe --> copy utilman.exe utilman1.exe & copy cmd.exe cmd1.exe
  9. Next, delete cmd.exe del cmd.exe
  10. renaming rename cmd.exe utilman.exe
  11. Reboot your system
  12. Enter the login screen and in the right bottom corner, click on the 'narrator' and your cmd.exe should open.
  13. Enter this command: net user (list users)
  14. Enter these commands below: net user *Username* *NewPassword*
  15. Login in your Account


How to reset the password on windows 10 (local acccount)
Using Installation Media

  1. Boot into the installation media
  2. Boot + F2, F5, F7, F8, F10, F12 or delete
  3. Once entered press shift + F10 or Next + Repair My Computer to open Command Prompt
  4. Enter this command: net user (list users)
  5. Enter these commands below: net user *Username* *NewPassword*
  6. Login in your Account


How to reset the password on windows 10 (local acccount)
Using build-in Administrator Account

  1. Boot in Recovery Mode or Installation Media
  2. Open Command Prompt (CMD)
  3. Enter this command: net user administrator /active:yes
  4. Reboot your Machine
  5. Login as Administrator


5 ways to reset/change your password

  1. Open Control panel --> User Accounts --> Click *user* --> Click 'change password' --> enter new password

    2. * This Only works if you remember the previous password *
  2. Settings --> Accounts --> Sign-in Option --> Click 'change' --> enter new password

  3. Open up Computer Management --> Local Users and Groups --> Users --> Right Click *User*
    --> Click Proceed (If popup is shown) --> Enter new password

  4. GUI + r --> enter netplwiz --> Choose User --> Click Reset Password --> Enter new password
  5. Using Command Prompt --> READ PREVIOUS METHODS
